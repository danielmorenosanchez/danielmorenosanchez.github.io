
# Daniel Moreno Sanchez | Professional Portfolio

Professional portfolio website created to present my academic background, technical skills, projects, and contact information as a Telecommunications Engineer interested in robotics, artificial intelligence, electronics, and web development.

## Overview

This project is a personal portfolio featuring an introduction, educational background, skills, highlighted projects, and contact links.

## Features

- Responsive web design for desktop, tablet, and mobile devices.
- Light and dark mode toggling.
- Professional introduction section.
- Organized skills and technologies section.
- Featured projects section.
- Contact section with social media icons.

## Sections

### Home

Includes a brief personal introduction, professional objective, and a call-to-action button to explore the projects.

### About Me

Includes educational background, areas of interest, relevant experience, and professional goals.

### Skills

Includes programming languages, software tools, technologies, and languages.

### Projects

Includes featured technical projects, such as:

- Low-cost 2.4 GHz Wi-Fi radar with signal attenuation capability.
- Autonomous vehicle controlled via a color selector.

### Contact

Includes links to email, LinkedIn, and GitHub.

## Technologies Used

- HTML5
- CSS
- JavaScript
