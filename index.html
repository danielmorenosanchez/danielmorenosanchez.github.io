Trabajo destacado

Proyectos

1

Radar Wi-Fi 2.4 GHz con opción de atenuación de señales de bajo costo

Desarrollo de un prototipo de bajo costo para detectar señales Wi-Fi mediante un microprocesador ESP32 y una antena modificada como antena direccional. El sistema permite captar señales y estimar su posición y distancia mediante el RSSI. También incorpora una opción para seleccionar una señal escaneada y atenuarla mediante un oscilador controlado por voltaje, que genera ruido en las frecuencias de los canales de Wi-Fi de 2.4 GHz.

Tecnologías utilizadas

  • C++
  • Electrónica analógica y digital
  • Wi-Fi 2.4 GHz
  • HTML
  • CSS
2

Vehículo autónomo controlado mediante un selector de colores

Desarrollo de un vehículo autónomo equipado con un sensor de color previamente calibrado para ejecutar tres acciones: avanzar, girar a la izquierda y girar a la derecha. La selección podía realizarse colocando bloques de color directamente o controlando el selector mediante Bluetooth.

Tecnologías utilizadas

  • Bluetooth
  • C++
  • MicroPython
  • Electrónica analógica y digital
